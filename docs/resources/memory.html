Memory

Memory is divided roughly into long-term and short-term (sensory memory and working memory, where are consciousness is happening). The long-term one can be subdivided into:

  1. Declarative (explicit) memory:

    1. Episodic memory (Events): Specific personal experiences from a particular time and place

      1. Autobiographical memory (Flashbulb memory)

      2. Experimental episodic memory

    2. Semantic memory (Facts): World/object/language knowledge, conceptual priming

  2. Non-declarative (implicit) memory

    1. Procedural memory: Skills (motor and cognitive)

    2. Perceptual representation system: Perceptual priming

    3. Classical conditioning: Conditioned responses between two stimuli

    4. Non-associative learning: Habituation sensitization
