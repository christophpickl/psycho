Language

  • Universals

    • Hierarchical structure: sentences; phonemes (vowel/consonant)

    • Grammar:

      • Rules on how to combine units for next level

  • Development

    • Tendency for over-generalization; e.g. "goed"

    • Coo/babble to prepare vocal appartus
